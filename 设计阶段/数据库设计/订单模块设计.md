# 订单模块设计

## 1. 引言

本文档描述了电商应用订单模块的数据库设计，包括订单表。

## 2. 订单表

| 字段名        | 数据类型     | 说明                             |
| ------------- | ------------ | -------------------------------- |
| id            | int          | 订单ID，主键，自增               |
| user_id       | int          | 用户ID，外键                     |
| goods_id      | int          | 商品ID，外键                     |
| quantity      | int          | 商品数量                         |
| price         | decimal      | 订单总价                         |
| status        | tinyint      | 订单状态（1：待支付，2：已支付，3：已发货，4：已收货）|
| created_at    | datetime     | 创建时间                         |
| updated_at    | datetime     | 更新时间                         |

## 3. 结论

本文档定义了电商应用订单模块的数据库设计，为开发团队和数据库管理员提供了数据存储和管理的依据。
